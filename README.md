# gateway-app - VueJS 3 APP with some Vuetify Elements

## Project setup

```bash
npm install
```

## Usage

In the `src/helpers/helper.js` file its the url for the `gateway-api`. Modify if change default port `3000`.

### Compiles and hot-reloads for development

```bash
npm run serve
```

### Compiles and minifies for production

```bash
npm run build
```

### Lints and fixes files

```bash
npm run lint
```
